Project Title
Integrated Financial Model & DCF Valuation – ITC Ltd
Project Summary
Developed an integrated three-statement financial model for ITC Ltd. using five years of historical financial data (FY2021–FY2025). Forecasted financial performance through FY2032 based on historical trends and business assumptions, and performed a Discounted Cash Flow (DCF) valuation to estimate the company's intrinsic value and assess whether the stock was overvalued or undervalued.
Business Objective
Estimate the intrinsic value of ITC Ltd. by forecasting future financial performance and valuing the business using the Discounted Cash Flow (DCF) method.
